Código-fonte / Pré formatado

O comando document.getElementById('Teste') é escrito em Javascript.

Aqui vamos vê como inserir código no meio de um texto abaixo temos o exemplo sem formatação apenas a TAG CODE.

num = int(input('Digite um número')) if num % 2 == 0; print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do código')

Já aqui o mesmo código porém de forma identada:

        
    num = int(input('Digite um número'))
    if num % 2 == 0;
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do código')
